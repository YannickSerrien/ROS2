# Course Resources

This directory contains reference materials, cheat sheets, and troubleshooting guides to support your learning throughout the course.

## Contents

### Cheat Sheets

Quick reference guides for common tasks:

- **`cpp-quick-reference.md`** - Modern C++ syntax and features
- **`ros2-commands.md`** - Essential ROS2 CLI commands
- **`cmake-reference.md`** - CMake and package.xml patterns
- **`debugging-guide.md`** - Debugging ROS2 C++ applications

### References

- **`references.md`** - Links to official docs, tutorials, books
- **`recommended-reading.md`** - Books and articles for deeper learning
- **`online-resources.md`** - Useful websites, forums, communities

### Troubleshooting

- **`troubleshooting.md`** - Common issues and solutions
- **`faq.md`** - Frequently asked questions
- **`error-messages.md`** - Decoding common error messages

### Best Practices

- **`best-practices.md`** - Coding standards and patterns
- **`code-style-guide.md`** - C++ and ROS2 style guidelines
- **`performance-tips.md`** - Optimization techniques

### Tools

- **`tools.md`** - Recommended development tools
- **`debugging-tools.md`** - GDB, Valgrind, profilers
- **`visualization-tools.md`** - RViz, PlotJuggler, rqt

### Glossary

- **`glossary.md`** - ROS2 and robotics terminology

## How to Use

Keep this directory bookmarked for quick reference while working through the course. When you encounter issues, check the troubleshooting guide first. Use the cheat sheets as quick references during development.
